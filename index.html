The Searcher

I made this application with python when I first started coding. I then learnt about git, and automatically published it there. Git is extremely useful not only for group projects but for one man projects! This app takes an input then opens up a browser and searches google for the input. It's very simple but helped me get an understanding of web scraping. I could have changed the way it searched by making it go to google.com and make it find the search box with driver.findElementById('q') which basically finds the searchbox by it's ID. Then it would type the search term in automatically. I found a much easier and faster way. Basically, I found how the Google search urls work. it goes: https://google.com/search+term. The search term is encoded into a url. That was another learning point. Discovering things for yourself.
